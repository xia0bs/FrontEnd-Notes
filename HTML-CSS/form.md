## form 标签

1. 表单标签

```
form action="a.php" method="post"
```

> action 提交的位置
> method 提交的方式
> get 信息量少 不安全
> post 信息量多 安全 

2. 输入框

```
用户名：input type="text" placeholder="请输入用户名" maxlength="10"
```

> type： 制定空间的类型  
> placeholder： 提示的文本  
> maxlenght:限制输入的最大字符  
> size：制定控件的宽度  
> name：和后台进行数据交互的  
> disabled：禁用  
> readonly:只读
